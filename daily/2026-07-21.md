## Repos

### [synthetic-sciences/openscience](../repos/synthetic-sciences__openscience.md)
- 一句話：把「科學研究整個 loop（讀 paper → 假設 → 寫 code → 跑實驗 → 分析 → 寫報告）」設計成一個 agent session 的本機 workbench。
- 綁 `127.0.0.1` + Host/Origin allowlist，**沒有 remote mode**；商業版 Atlas 只是 optional client。
- Prompt 分兩層組（provider-level system + agent-level workflow），讓「換 model」和「換 agent 角色」變成正交軸。
- Skill catalog 兩路 load 同 interface（cache-from-cloud vs bundled-source），開發跑 source 不用 mock 雲端，發布版又能拿最新。

### [agentlas-ai/Agentlas-OS](../repos/agentlas-ai__Agentlas-OS.md)
- 一句話：用三個 meta-agent（single-builder / team-builder / packager）產出 runtime-neutral 的 agent 包，canonical core 只有 markdown + JSON contract，各 runtime（Claude Code / Codex / Gemini）只是薄 adapter。
- `.agentlas/super-ontology-*.json` × 25+：每個治理維度（coverage / consensus / causal-impact / adversarial-provenance / ...）獨立一支 contract file。
- Skill 有 lifecycle：candidate → trial evidence → curator decision，runtime first-class recall 預設關掉。
- Ontology runtime 走 SQLite + FTS5 + deterministic local vectors + GraphRAG，缺 parser 記 `unsupported_pending_adapter` 而非假裝成功。
